A simple client side mod to bring back the original look of baby mobs prior to the 26.1 baby mob overhaul.

This mod restores the original baby models of the following mobs:

- Pigs
- Sheep
- Chickens
- Cows
- Mooshrooms
- Wolves
- Cats
- Ocelots
- Horses
- Donkeys
- Mules
- Skeleton Horses
- Zombie Horses

The models of baby rabbits are unchanged as their default model was updated 

Compatible with resource packs that change mob textures

Releases available on modrinth: https://modrinth.com/project/old-babies
